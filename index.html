AsyncIO Any to Server (A2S) Query Library¶

This library is an asyncio implementation of the Any to Server (A2S) query protocol for querying information from Goldsource and Source Engine servers. It provides a simplistic client interface that attempts to handle the nuances of querying different engine and game versions while allowing users to perform multiple concurrent queries.
