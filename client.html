Query Client¶

class aioa2squery.A2SQueryContext(*, game_engine: aioa2squery.enumerations.engine_versions.Engine = <Engine.SOURCE: 2>, app_id: Optional[int] = None, packet_split_size: Optional[int] = 1248, timeout: Optional[float] = 10.0, use_compression: bool = False)¶

Query client that handles the nuances of querying different Source engine and game versions

Parameters

  • timeout (float) – Default timeout duration (in seconds) when making server queries

  • game_engine (aioa2squery.Engine) – Explicitly treat all query responses as being in the response format for a specific game engine version

  • app_id (int) – If specified treat all responses as being in a format for this explicit Steam application ID

  • packet_split_size (int) – How big to make query before splitting query request into additional packets. Steam uses a packet size of 1400 bytes (the default).

  • use_compression (bool) – Whether or not to apply compression on multi-packeted query requests

Raises

RuntimeError – If query client is initialized outside of an event loop/coroutine function

async query_info(host: Optional[str] = '127.0.0.1', port: Optional[int] = <QueryPort.SRCDS: 27015>, timeout: Optional[float] = None) → Tuple[aioa2squery.responses.A2SInfoResponse, int]¶

Retrieve information about the server including, but not limited to: its name, the map currently being played, and the number of players

Note

Rag Doll Kung Fu servers always return ServerType.UNKNOWN for aioa2squery.A2SInfoResponse.server_type in the response.

Note

HLTV servers may respond with a late request that is either incomplete or incorrect if the number of slots is 255.

Parameters

  • host – Host of the server to query

  • port – Port of server to query

  • timeout – Timeout duration (in seconds) when making server queries

Raises

  • asyncio.TimeoutError – If query takes exceeds timeout duration

  • aioa2squery.ResponseError – If there is a problem completing, reading or parsing the response data

Returns

Basic information about the server and the estimated ping to the server in milliseconds

Return type

tuple[A2SInfoResponse, int]

async query_players(host: Optional[str] = '127.0.0.1', port: Optional[int] = <QueryPort.SRCDS: 27015>, timeout: Optional[float] = None) → Tuple[aioa2squery.responses.A2SPlayersResponse, int]¶

Retrieve information about the players currently on the server.

Note

SourceTV does not respond to this query.

Note

When a player is trying to connect to a server, they are recorded in the number of players. However, they will not be in the list of player information.

Warning

Counter-Strike: Global Offensive Dedicated Server by default returns only max players and server uptime. You have to change server CVar "host_players_show" in server.cfg to value "2" (show full info) if you want to revert to old format with players list.

Parameters

  • host – Host of the server to query

  • port – Port of server to query

  • timeout – Timeout duration (in seconds) when making server queries

Raises

  • asyncio.TimeoutError – If query takes exceeds timeout duration

  • aioa2squery.ResponseError – If there is a problem completing, reading or parsing the response data

Returns

Details about each player on the server and the estimated ping to the server in milliseconds

Return type

A2SPlayersResponse
